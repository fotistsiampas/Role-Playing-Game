# Introduction
Σκοπός του μαθήματος είναι η σχεδίαση και ανάπτυξη ψηφιακού παιχνιδιού στο unity σε 2D RPG. Η επεξεργασία και η εξέλιξη του παιχνιδιού γίνεται σύμφωνα με μεθοδολογικά πλαίσια που είναι τα tutorials από το Ruby's Adventure,ένα 2D RPG video game και υπάρχουν κάποια επιπρόσθετα ζητούμενα.
# Summary
Ο ήρωας του παιχνιδιού είναι ένας νίντζα ο οποίος είναι μέσα σε μία μικρή αρένα σκοπός του είναι να αποφύγει την παγίδα που υπάρχει έτσι ώστε να μην χάσει ζωή και δεν πρέπει να σκοτωθεί από τα ρομπότ που υπάρχουν μέσα στον χώρο για βοηθό του έχει την γάτα του η οποία έχει καταλάβει τα ρομπότ που υπάρχουν και του δίνει εντολή να τα σκοτώσει. Μπορεί να τηλεμεταφέρεται μέσω ενώς σήματος κεραυνού που υπάρχει στο έδαφος και εκτός από τα ρομπότ πρέπει να συλλέγει και κάποια shuriken για να αυξάνονται οι πόντοι του.
